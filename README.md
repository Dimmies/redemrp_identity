# xrp_identity
 a Multicharacter and Identity system for xRP Framework

## 1. Installation
- Be sure you have XRP Installed
if not -> https://github.com/amakuu/xrp/
- Clone xrp_identity into [xrp] folder
- add ```ensure xrp_identity``` after ```ensure xrp```
- Have fun with changing camera coords ! :D

## 2. Usage
- Any config you can set in config.lua file
